# 📓 Movie App
Movie-App adalah aplikasi pencarian film global yang memungkinkan pengguna untuk menemukan semua jenis film dari seluruh dunia — mulai dari film klasik hingga rilisan terbaru, dari berbagai negara.✨

- ✅ Trending Movies
